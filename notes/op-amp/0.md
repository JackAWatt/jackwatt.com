* Inv amp

$$A_V = \frac {-RF}{R1}$$


* non-inv

$$A_V = 1 + \frac {R_f}{R_i}$$

$$B = \frac {R_i}{R_i + R_f}$$

$$R_f = 0 && R_I = \infinity \then A_V = 1$$

* fmax - unity gain

$$fmax = \frac {UGBW}{A_V}$$

* 2 opamp "law"

$$V_+ = V_- = 0V$$

$$I_+ = I_- = 0mA$$

* fmax slewrate

$$fmax = \frac{SR}{2\pi V_{opk}}$$

* summer

$$V_O = V_1 * \frac {rf}{r1}  * V_2 * \frac {rf}{r2} .. V_N * \frac {rf}{r_n}$$

* CMRR

$$CMRR = \frac {A_{ol}}{A_{cm}}$$

$$CMRR = 20 log \frac {A_{ol}}{A_{cm}}$$
