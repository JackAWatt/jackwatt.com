# DC Analysis

* $V_B = Vcc * \frac {R2}{R1+R2}$

* $V_E = V_B - 0.7v$

* $I_C = \frac {V_E}{R_E}$

* $V_{CE} = Vcc - I_C(R_C + R_B)$

* $V_C = Vcc - I_CR_C$

* $I_{CSAT} = \frac {Vcc}{R_C+R_B}$

# AC Analysis

* $r_e = \frac {25mV}{I_E}$

* $A_v = \frac {R_c{//}R_L}{r_e+R_{E1}}$

* $V_O = V_i * A_v$

* $R_in = R_1{//}R_2{//}\beta(r_e+R_{E1})$
